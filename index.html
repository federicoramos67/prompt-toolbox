ELEGÍ MODO DE INTERACCIÓN
¿Cómo querés usar este prompt?
Cada modo afecta la calidad y el idioma de las respuestas. El modo híbrido es el recomendado para hispanohablantes porque combina la calidad del prompt original en inglés (como fue diseñado por Anthropic) con respuestas naturales en español.
ℹ️ NOTA SOBRE IDIOMAS
Los modelos de IA razonan mejor en inglés porque fueron entrenados predominantemente en ese idioma. Los prompts oficiales de Anthropic también están en inglés. Traducir el prompt al español puede reducir sutilmente la calidad del output.
🇺🇸
Oficial — English
Usa el prompt original exacto de Anthropic. Escribís y recibís respuestas en inglés. Máxima calidad y fidelidad al diseño original.
Tu input EN → Prompt EN → Respuesta EN
🇪🇸
Traducido — Español
Usa el prompt traducido al español. Todo en español pero puede tener leve pérdida de calidad porque el modelo fue optimizado para prompts en inglés.
Tu input ES → Prompt ES → Respuesta ES
🌐
Híbrido — Hispano Smart
Escribís en español, el modelo usa el prompt oficial en inglés para razonar internamente con máxima calidad, y te devuelve la respuesta final en español. Lo mejor de ambos mundos.
Tu input ES → traducción invisible → Prompt EN → razonamiento EN → traducción invisible → Respuesta ES
